Tutorial: Estruturas de Repetição em JavaScript

Introdução

Estruturas de repetição são fundamentais para realizar tarefas que envolvem repetição de código. Em JavaScript, temos três principais tipos de repetição: for, while e do...while.

Estrutura for

A estrutura for em JavaScript é uma das formas mais utilizadas para executar repetidamente um bloco de código, quando sabemos o número exato de iterações que precisamos fazer. Esse tipo de laço é ideal para cenários onde temos uma condição bem definida e controlada, como contar até um número específico ou iterar sobre uma coleção de itens.

for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Este código imprime os números de 1 a 10.

Estrutura while

A estrutura de repetição while é um dos tipos mais comuns de laços de repetição em muitas linguagens de programação. Ela é usada para executar um bloco de código repetidamente enquanto uma condição for verdadeira. Ou seja, enquanto a condição especificada no início for satisfeita, o código dentro do bloco será executado, e quando a condição se tornar falsa, o laço será encerrado.

let i = 1;
while (i <= 10) {
    console.log(i);
    i++;
}

Este código imprime os números de 1 a 10 utilizando o while.

Estrutura do...while

A estrutura de repetição do...while é muito similar ao while, mas com uma diferença importante no momento em que a condição de repetição é verificada. A principal característica do do...while é que ele executa o bloco de código pelo menos uma vez, independentemente de a condição ser verdadeira ou falsa inicialmente. Após a primeira execução, a condição é verificada, e, se for verdadeira, o bloco de código será executado novamente.

let numero;
do {
    numero = parseInt(prompt('Digite um número maior que 5:'));
} while (numero <= 5);

Este código vai pedir para o usuário digitar um número até que ele seja maior que 5.

Exercícios

  • Crie um código que liste todos os números de 1 a 100, mas somente os números divisíveis por 5.
  • Escreva um código que peça para o usuário inserir um número até que ele insira um número ímpar.
  • Crie um código que continue pedindo um número até que o usuário insira um número múltiplo de 3.
